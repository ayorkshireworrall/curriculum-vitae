\section{Projects}

\projectHeadingWithDateNoLink{GLA OPS}{Java, Spring, AngularJS, Postgres, AWS, Selenium}{Jan 2021 - Jan 2022}
Senior developer on an agile project to produce grant application and grant distribution system for the Greater London Authority. Allows external applicants to apply for programmes that are highly customisable by internal users. Applicant projects are fully audited and version controlled with automated payments based on approval criteria. System built to be robust as there are nearly 3000 organisations who have requested £10bn in funding, of which £6bn has been granted and paid via the application.\\
\sectionsep
  
\projectHeadingWithDateNoLink{NE OPS}{Java, Spring, AngularJS, Postgres, AWS, Selenium}{Oct 2020 - Jan 2021}
An OPS implementation using the GLA OPS open source code. Done as a solo project for a smaller client (North of Tyne Combined Authority). Was required to set up AWS infrastructure, deployment pipeline and configure code to rebrand the application. Specific requirements also required a few code changes to meet their business needs.
\sectionsep

\projectHeadingWithDateNoLink{Plan Zheroes}{NodeJS, ReactJS, Postgres, AWS, Kubernetes}{Apr 2020-Oct 2020}
A pro bono legacy project to establish a network of volunteers distributing excess food from businesses to charities who required it. Had to migrate the project from broken infrastructure to a Kubernetes cluster and configure a new build pipeline. Did a security audit of vulnerable APIs and fixed existing defects in both the front end ReactJS code and backend NodeJS code.
\sectionsep

\projectHeadingWithDateNoLink{GRP2}{Java, Liferay, OSGI, MySQL, ReactJS, Selenium}{Jan 2018-Apr 2020}

Responsible for implementing the automated e2e regression testing using selenium, managing OSGI dependencies and built an automated process to provision sites, organisations and users based on csv inputs.
